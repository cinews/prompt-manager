# 프롬프트 매니저 앱 (Prompt Manager App)

이 문서는 프롬프트 매니저 애플리케이션의 구조, 기능 및 실행 방법에 대한 가이드를 제공합니다.

## 프로젝트 개요
프롬프트 매니저 앱은 사용자가 AI 프롬프트를 효율적으로 저장, 관리, 검색할 수 있도록 돕는 React 애플리케이션입니다.

## 주요 기능

1. **프롬프트 관리 (CRUD)**
   - **생성 (Create)**: '새 프롬프트' 버튼을 통해 제목, 내용, 태그, 카테고리를 입력하여 새로운 프롬프트를 등록할 수 있습니다.
   - **조회 (Read)**: 메인 화면에서 등록된 프롬프트 목록을 카드 형태로 확인할 수 있습니다. 카드 클릭 시 상세 모달이 열립니다.
   - **수정 (Update)**: 프롬프트의 내용을 수정할 수 있습니다.
   - **삭제 (Delete)**: 더 이상 필요하지 않은 프롬프트를 삭제할 수 있습니다.

2. **이모티콘 및 UI 한글화**
   - 앱의 주요 인터페이스(버튼, 타이틀, 알림 등)가 한국어로 제공됩니다.

3. **즐겨찾기 (Favorites)**
   - 프롬프트에 '좋아요(하트)'를 표시하여 즐겨찾기에 추가할 수 있습니다.
   - 사이드바의 'Favorites' 메뉴를 통해 즐겨찾기 된 프롬프트만 모아볼 수 있습니다.

4. **카테고리 및 필터링**
   - 사이드바에서 카테고리별로 프롬프트를 필터링하여 볼 수 있습니다.
   - 카테고리별 프롬프트 개수가 표시됩니다.

5. **검색 (Search)**
   - 키워드를 통해 제목, 내용, 태그에서 프롬프트를 검색할 수 있습니다.

## 기술 스택
- **Frontend**: React (Vite)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React

## 프로젝트 구조 (주요 파일)
- `src/App.jsx`: 메인 애플리케이션 로직, 라우팅(필터링), 상태 관리가 포함되어 있습니다.
- `src/components/Sidebar.jsx`: 카테고리 선택 및 검색 기능을 담당하는 사이드바 컴포넌트입니다.
- `src/components/PromptList.jsx` & `PromptCard.jsx`: 프롬프트 목록과 개별 카드를 렌더링합니다.
- `src/components/PromptForm.jsx`: 프롬프트 생성 및 수정을 위한 폼 컴포넌트입니다.
- `src/components/PromptDetailModal.jsx`: 프롬프트의 상세 내용을 보여주는 모달입니다.

## 시작하기 (Getting Started)

1. 의존성 설치:
   ```bash
   npm install
   ```

2. 개발 서버 실행:
   ```bash
   npm run dev
   ```
